## Install
### MacOSX
(as root)
  <pre>
  git clone https://github.com/abaykan/53R3N17Y.git /usr/local/share/serenity
  >echo 'alias serenity="/usr/local/share/serenity && ./serenity"' > ~/.zshrc
  cd /usr/local/share/serenity
  pip install -r requirements.txt
  serenity -h</pre>

### Linux
(as root)
  <pre>
  git clone https://github.com/abaykan/53R3N17Y.git /usr/local/share/serenity

  serenity -h</pre>
